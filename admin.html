How to use this panel

  1. Edit content using the editors in each section.
  2. Click Save Draft to store changes without going live.
  3. Click Preview to see draft changes on the live site.
  4. Click Publish to make changes visible to everyone.
  5. Use Reset to Default in Settings to undo all changes.
